## 🔧 基础设施

### Docker 环境 ✅

**配置文件**：
- `env/docker-compose.yml` - 服务编排
- `env/Dockerfile.backend` - 后端镜像
- `env/Dockerfile.frontend` - 前端镜像

**管理脚本**：
- `env/start.sh` - 启动服务
- `env/stop.sh` - 停止服务
- `env/check.sh` - 代码检查（临时容器）
- `env/check-running.sh` - 代码检查（运行中容器）

**特点**：
- ✅ 一键启动
- ✅ 热重载
- ✅ Volume 持久化
- ✅ 健康检查
- ✅ 国内镜像优化

### CI/CD 流水线 ✅

**GitHub Actions**：
- Backend Check (Docker) - Black、isort、Ruff、pytest
- Frontend Check (Docker) - Prettier、ESLint、TypeScript、Build
- Docker Build Check - 镜像构建验证
- Integration Tests - docker compose 集成测试

**特点**：
- ✅ 全部在 Docker 中执行
- ✅ 与本地环境 100% 一致
- ✅ 自动化测试和检查

**Pre-commit Hooks**：
- trailing-whitespace, end-of-file-fixer
- check-yaml, check-json
- Black, isort, Ruff (后端)
